# ESP32-Sprinkler-System
A low-cost smart sprinkler system built using ESP32, ESPHome, and Home Assistant. Control and automate your irrigation system over WiFi using relay modules and an existing 24VAC sprinkler setup

✨ Features
📱 Phone control via Home Assistant
⏱️ Automated watering schedules
⚡ ESPHome-based firmware
💸 Budget-friendly (~$10 build)
🔌 Works with existing 24VAC sprinkler systems
